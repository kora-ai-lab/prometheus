# Semaine 1 : Design System & Avatar Prometheus

## 🎯 Objectif de la Semaine
**Créer les fondations visuelles de Prometheus - l'identité qui rendra l'agent réel et inspirant**

---

## 📅 Planning Journalier

### 🌅 Lundi - Jour 1 : Fondations Visuelles
#### 🎨 Design System - Base
- [ ] **Définir palette Prometheus**
  - [ ] Couleur primaire : Feu sacré (#FF6B35)
  - [ ] Couleur secondaire : Doré (#FFD700) 
  - [ ] Couleur tertiaire : Bleu nuit (#1A1A2E)
  - [ ] Couleurs neutres : Gris, blanc, noir
  - [ ] Couleurs d'état : Vert succès, rouge erreur, orange attention

- [ ] **Créer typographie hiérarchique**
  - [ ] Titres : Poppins Bold (32px, 24px, 18px)
  - [ ] Sous-titres : Poppins SemiBold (16px, 14px)
  - [ ] Corps : Inter Regular (16px, 14px)
  - [ ] Code : JetBrains Mono (14px, 12px)
  - [ ] Tags : Inter Medium (12px)

- [ ] **Définir spacing system**
  - [ ] Base : 8px grid
  - [ ] Scale : 8, 16, 24, 32, 48, 64, 96px
  - [ ] Margin/Padding : 4, 8, 12, 16, 24, 32px

#### 🎨 Avatar - Concept Initial
- [ ] **Recherche visuelle**
  - [ ] Collecter références : Cristaux, flammes, formes organiques
  - [ ] Étudier avatars IA existants (Claude, ChatGPT, etc.)
  - [ ] Définir forme de base : Cristalline avec cœur de flamme
  - [ ] Sketch concepts (3-4 variations)

### 🌞 Mardi - Jour 2 : Design System Avancé
#### 🎨 Design System - Complétion
- [ ] **Définir shadows et elevations**

- [ ] **Créer transitions et animations**

- [ ] **Composants UI de base**

### 🌙 Mercredi - Jour 3 : Avatar Animé
#### 🤖 Avatar - 4 États Émotionnels
- [ ] **État 1 : Écoute (Respiration calme)**
  - [ ] Animation : Respiration douce 4s cycle
  - [ ] Couleur : Bleu nuit doux
  - [ ] Lumière : Pulse lent
  - [ ] Forme : Stable et ouverte

- [ ] **État 2 : Réflexion (Circuits actifs)**
  - [ ] Animation : Circuits lumineux 2s cycle
  - [ ] Couleur : Bleu nuit + touches dorées
  - [ ] Lumière : Pulses rapides
  - [ ] Forme : Concentrée

- [ ] **État 3 : Illumination (Éclats dorés)**
  - [ ] Animation : Éclats spontanés
  - [ ] Couleur : Doré dominant
  - [ ] Lumière : Éclats intenses
  - [ ] Forme : Expansion temporaire

- [ ] **État 4 : Co-création (Danse fluide)**
  - [ ] Animation : Mouvement organique 3s cycle
  - [ ] Couleur : Feu sacré + doré
  - [ ] Lumière : Flux continu
  - [ ] Forme : Dynamique et expressive

#### 🤖 Avatar - Micro-interactions
- [ ] **Suivi oculaire**
  - [ ] Mouvement yeux vers curseur
  - [ ] Clignement naturel
  - [ ] Expressions subtiles

- [ ] **Réactions**
  - [ ] Réponse hover : Légère inclinaison
  - [ ] Réponse click : Pulse de reconnaissance
  - [ ] Réponse erreur : Vibration subtile
