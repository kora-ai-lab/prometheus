# T5.4 Documentation Design

> **Design Document** - User guides in English and French

## Overview

Write comprehensive user guides (EN/FR) covering installation, usage, and troubleshooting.

## Content

### 1. README.md (English)
- Quick start
- Installation (curl | sh)
- Usage examples
- API keys setup

### 2. README.fr.md (French)
- Same content translated to French

### 3. TROUBLESHOOTING.md
- Common issues and solutions

## Structure

```
docs/
├── README.md       # English
├── README.fr.md   # French
└── TROUBLESHOOTING.md
```

## Acceptance Criteria

- [ ] README.md in English
- [ ] README.fr.md in French
- [ ] TROUBLESHOOTING.md
