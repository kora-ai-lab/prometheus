## Trace ID & Spans

### Trace Context

```
TraceID:   32-char hex (128-bit)
SpanID:    16-char hex (64-bit)  
ParentSpanID: 16-char hex (64-bit)
```

### Span Attributes

| Attribute | Type | Description |
|-----------|------|-------------|
| trace_id | string | Full trace ID |
| span_id | string | This span ID |
| parent_span_id | string | Parent span ID |
| start_time | int64 | Unix timestamp ns |
| end_time | int64 | Unix timestamp ns |
| duration_ms | float64 | Elapsed time |
| operation_name | string | Name of operation |
| status_code | int | 0=OK, 1=Error |
| error.message | string | Error message if any |

### Custom Attributes per Span

| Key | Type | Purpose |
|-----|------|---------|
| prometheus.task_id | string | Task ID |
| prometheus.task_type | string | task, llm_call, exec |
| prometheus.success | bool | Task success |
| http.method | string | HTTP method |
| http.url | string | URL |
| http.status_code | int | Response code |
| llm.model | string | LLM model used |
| llm.tokens_used | int | Total tokens |
| llm.latency_ms | float | LLM call latency |
| exec.command | string | Command executed |
| exec.exit_code | int | Exit code |

## Integration Points

### Task Loop

```go
func (t *Task) Run(ctx context.Context) {
    span := tracer.StartSpan("task.run",
        "task_id", t.ID,
        "task_type", t.Type)
    defer span.End()
    
    // ... execution
    span.SetAttribute("success", true)
}
```

### LLM Client

```go
func (c *LLM) Complete(ctx context.Context, prompt string) {
    span := tracer.StartSpan("llm.complete",
        "model", c.model,
        "prompt_tokens", countTokens(prompt))
    defer span.End()
    
    resp, err := c.call(ctx, prompt)
    if err != nil {
        span.SetStatus(1, err.Error())
    }
    span.SetAttribute("completion_tokens", resp.Usage...)
}
```

### Executor

```go
func (e *Executor) Execute(ctx context.Context, cmd string) {
    span := tracer.StartSpan("exec.execute",
        "command", cmd)
    defer span.End()
    
    result := e.run(cmd)
    span.SetAttribute("exit_code", result.ExitCode)
}
```
