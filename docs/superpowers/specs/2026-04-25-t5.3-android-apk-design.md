# T5.3 Android APK Design

> **Design Document** - Android WebView wrapper APK

## Overview

Create Android WebView wrapper that loads the web UI.

## Approach

Since we've built the web UI already (T5.1), the Android APK is simply:
- Android app with WebView
- Loads http://localhost:8080 (where Go server runs)
- Minimal Java/Kotlin to host WebView

## Architecture

```
android/
├── app/
│   ├── src/main/
│   │   ├── java/.../MainActivity.kt
│   │   └── res/layout/activity_main.xml
│   └── AndroidManifest.xml
```

## Note

This is a simple wrapper. Users need to:
1. Install the APK on Android
2. Run `prometheus --web` on a server
3. Access via WebView

## Acceptance Criteria

- [ ] AndroidManifest.xml with WebView permission
- [ ] MainActivity.kt loads URL
- [ ] Build produces APK (optional - requires Android SDK)
