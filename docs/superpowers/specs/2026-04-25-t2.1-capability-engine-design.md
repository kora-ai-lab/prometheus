## 1. Overview

Expand the Capability Engine to support 60+ tools with intelligent lookup: cache → registry → package managers → internet → Forge.

## 2. Architecture

Two sub-parts working in parallel:

### Sub-part A: Registry Expansion
```go
// Categories:
Dev      : git, python3, python, node, npm, go, rust, java, maven, gradle
Web      : chromium, firefox, curl, wget, httpie
Data     : jq, sqlite3, redis-cli, csvkit, pandas, requests
DevOps   : docker, kubectl, terraform, ansible-playbook
Security : semgrep, nmap, trivy, burp
Media    : ffmpeg, imagemagick, yt-dlp, whisper
Mobile   : adb, apktool, dex2jar
Vision   : scrot, import, tesseract
Forge    : python3, bash (always present for forging)

// Capability struct
type Capability struct {
    Name         string            // unique name
    CheckCmd    string            // command to verify installation
    InstallCmds map[string]string // os/package manager → install command
    Type        string            // "system"|"pip"|"npm"|"cargo"|"forged"
    SizeMb      int               // approximate size
    Description string            // what the tool does
}
```

### Sub-part B: Ensure() Flow
```go
func (e *Engine) Ensure(ctx context.Context, name string) error {
    // 1. Check cache (SQLite) → already installed? → return nil
    // 2. Check registry → known tool? → install via appropriate package manager
    // 3. Try package managers: apt, pip, npm, brew, winget, cargo
    // 4. Try internet: PyPI, npm, crates.io, GitHub releases
    // 5. FORGE → if all fail, LLM generates tool code
}
```

## 4. Ensure() Priority Order

1. **Cache** (SQLite `capabilities_cache` table)
   - Check if already installed in this session
   - Return immediately if found

2. **Registry** (in-memory map)
   - Known tools with install recipes
   - Install via appropriate package manager

3. **Package Managers**
   - Try `apt search`, `pip search`, `npm search`, `brew search`, `winget search`
   - Parse output to find package name
   - Install found package

4. **Internet Search**
   - PyPI: `https://pypi.org/pypi/{package}/json`
   - npm: `https://registry.npmjs.org/{package}`
   - crates.io: `https://crates.io/api/v1/crates/{package}`
   - GitHub: `https://api.github.com/repos/{owner}/{repo}/releases/latest`

5. **FORGE** (T2.2 - future)
   - If nothing found, call LLM to generate tool
   - Save as "forged" capability

## 5. Storage

```sql
-- capabilities_cache table
CREATE TABLE capabilities_cache (
    name TEXT PRIMARY KEY,
    installed BOOLEAN DEFAULT 0,
    version TEXT,
    install_method TEXT,  -- registry, pip, npm, forge
    installed_at TEXT,
    metadata_json TEXT
);
```
