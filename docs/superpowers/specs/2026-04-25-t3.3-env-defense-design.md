## Part 1: Port Scanner

### Features

- Scan common ports (1-1024)
- Identify service by port number
- Detect suspicious ports (malwarecommonly used)
- Report open ports with risk level

### Risk Levels

| Risk | Ports |
|------|-------|
| High | 23 (telnet), 135-139 (NetBIOS), 445 (SMB), 3389 (RDP) |
| Medium | 21 (ftp), 23 (telnet), 110 (pop3), 143 (imap) |
| Low | 80, 443, 22, 3000, 8080 |

---

## Part 2: Permission Scanner

### Features

- Check file permissions (executable, writable, readable)
- Check for world-writable files in sensitive dirs
- Check for SUID/SGID files
- Check .ssh and .aws directories

### Findings

| Finding | Severity |
|--------|---------|
| World-writable /etc/passwd | Critical |
| World-writable sensitive files | High |
| SUID scripts | High |
| Unauthorized SSH keys | Critical |

## Part 3: DAST (Dynamic Application Security Testing)

### HTTP Checks

| Check | Description |
|-------|-------------|
| Server Headers | Check for information disclosure (server: Apache 2.4) |
| Security Headers | CSP, X-Frame-Options, HSTS |
| HTTP Methods | OPTIONS, TRACE enabled |
| Status Codes | 200 for not-found paths |
| Response Time | Slow responses (>3s) |

### TLS Checks

| Check | Description |
|-------|-------------|
| Certificate Validity | Expired, validity period |
| Certificate Chain | Root CA trust |
| Protocol Versions | TLS 1.0/1.1 deprecated |
| Cipher Suites | Weak ciphers |

### Endpoint Fuzzing

| Wordlist | Endpoints |
|----------|-----------|
| admin | /admin, /admin/, /administrator |
| api | /api, /api/v1, /api/v2 |
| config | /config, /configuration |
| backup | /backup, /backups, /dump |
| login | /login, /signin |
| debug | /debug, /debugger |

### Vulnerabilities Check

| Check | Severity |
|-------|----------|
| Directory Listing | Information disclosure |
| Default credentials | Authentication bypass |
| SQLi in query params | SQL injection |
| XSS in response | Cross-site scripting |
| SSRF in redirects | Server-side request forgery |
