# T5.2 Distribution Design

> **Design Document** - curl | sh installer with OS/Arch detection

## Overview

Create `curl | sh` installer one-liner with automatic OS/Arch detection.

## Components

### 1. Install Script (install.sh)
```bash
#!/bin/bash
set -e

# Detect OS
OS="$(uname -s | tr '[:upper:]' '[:lower:]')"
ARCH="$(uname -m)"

# Map arch
case "$ARCH" in
  x86_64) ARCH="amd64" ;;
  arm64|aarch64) ARCH="arm64" ;;
esac

# Download
URL="https://github.com/kora-ai-lab/prometheus/releases/latest/download/prometheus-${OS}-${ARCH}"

curl -fsSL "$URL" -o /usr/local/bin/prometheus
chmod +x /usr/local/bin/prometheus

echo "Installed prometheus to /usr/local/bin/prometheus"
```

### 2. Release Build Script
```bash
#!/bin/bash
# Build for all platforms
PLATFORMS="darwin-amd64 darwin-arm64 linux-amd64 linux-arm64 windows-amd64"

for plat in $PLATFORMS; do

done
```
