# T5.1 Web UI Design

> **Design Document** - Vanilla HTML/CSS/JS Web UI with integrated Go server

## Overview

Full-featured web UI: task input/output, metrics dashboard, settings panel.

## Existing Foundation

- `internal/ui/webui.go` - HTTP server (keep)
- `assets/static/` - static files (expand)

## Components

### 1. Task Panel
- Goal input textarea
- Submit button
- Output display (terminal-style)
- Task history sidebar

### 2. Metrics Dashboard
- RAM usage (from metrics collector)
- Tasks started/done
- LLM calls count
- Latency histogram display

### 3. Settings Panel
- API key input (masked)
- Model selection dropdown
- Save/Load config

### 4. API Endpoints
- `POST /api/execute` - execute task
- `GET /api/metrics` - return metrics JSON
- `POST /api/settings` - save settings

## Architecture

```
assets/static/
├── index.html    # Main UI
├── app.js      # Frontend logic
├── style.css   # Styling

internal/ui/
├── webui.go    # Server (existing, add API routes)
```
