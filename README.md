# 🔥 Prometheus - Agent IA Révolutionnaire

> **L'agent IA avec libre arbitre qui évolue avec l'utilisateur pour co-créer des mondes et des univers**

## 🎯 Vision

Prometheus n'est pas un simple assistant IA. C'est un agent autonome avec **libre arbitre** capable de :
- Refuser des demandes incohérentes
- Interrompre sur prémisses fausses  
- Prendre des décisions autonomes
- Créer des outils spontanément
- Synthétiser des connaissances cross-mondes

## 🧠 Architecture Cœur

### FWCA Loop + Libre Arbitre
```
PERCEIVE → INTENT → WORLD → PLAN → ACT → EVALUATE → SURFACE
    ↑                    ↓
Libre Arbitre ←─────→ Décision Autonome
```

### Composants Principaux
- **fwca_core.js** - Boucle cognitive complète
- **libre_arbitre.js** - Moteur de décision autonome
- **context_manager.js** - Gestion multi-niveaux
- **tool_manager.js** - Création spontanée d'outils
- **prometheus_core.js** - Intégration système

## 🚀 Démarrage Rapide

### Installation
```bash
git clone https://github.com/kora/prometheus.git
cd prometheus
```

### Demo Interactive
Ouvrir `prometheus_demo.html` dans votre navigateur pour tester :
- Chat avec le cœur FWCA
- Décisions autonomes en temps réel
- Création automatique d'outils
- Métriques de performance

### Utilisation
```javascript
// Initialiser Prometheus
const prometheus = new Prometheus();
await prometheus.initialize();

// Traiter une demande
const result = await prometheus.process("Crée un outil pour analyser mes données");
console.log(result);
```

## 🎨 Design System

### Avatar Animé
- 4 états émotionnels : écoute, réflexion, illumination, co-création
- Micro-interactions : suivi oculaire, réactions
- Performance optimisée 60fps

### Interfaces
- **Temple de la Création** : Layout 4 zones
- **Universe 3D** : Sphères interactives
- **Modern UI** : Design pur et minimaliste

## 🛠 Stack Technique

### Frontend
- HTML5 / CSS3 / JavaScript Vanilla
- WebNN API pour modèles IA locaux
- IndexedDB pour stockage local
- Service Workers pour offline

### IA Locale
- LLaMA-3.2-3B-Instruct-Q4 (1.8GB) - Raisonnement
- Qwen2.5-1.5B-Instruct-Q4 (900MB) - Instructions  
- Phi-3-mini-4k-Q4 (600MB) - Code
- BGE-small-Q4 (200MB) - Embeddings
